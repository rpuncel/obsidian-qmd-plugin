# Callout Blocks

Each callout below should render as a styled box with a colored left border, icon, and bold title.

::: {.callout-note}
This is a note. It should have a blue left border and an ℹ icon.
:::

::: {.callout-warning}
This is a warning. It should have a yellow/amber left border and a ⚠ icon.
:::

::: {.callout-important}
This is important. It should have a red left border and a ✖ icon.
:::

::: {.callout-tip}
This is a tip. It should have a green left border and a ✔ icon.
:::

::: {.callout-caution}
This is a caution. It should have an orange left border and a 🔥 icon.
:::

## Callout with custom title

::: {.callout-note title="Custom Title Here"}
This note has a custom title instead of the default "Note".
:::

## Callout with multi-paragraph body

::: {.callout-warning title="Watch Out"}
First paragraph of the warning body.

Second paragraph — the callout body should contain both paragraphs.
:::
